在 github 上写博客
================
github 上出品了 github pages 这个东东，写博可用的。需进行一些设置：

在 github 上 ：
* 新生成一个 repository, 名为 username.github.io

在我的 linux 机器(ubuntu 16.04)上：
* $ sudo apt-get install ruby-full
* $ sudo gem install jekyll bundler
