## Assert

开发工作完成后，设置preprocess 预处理 NDEBUG，可以将所有的 assert 语句都去掉。
注意：assert 中应该只判断不要操作对象。

    class String
    {
    public:
        enum {DEFAULT_SIZE = 8};
        char *d_array_p;
        int d_size;
        int d_length;
    public:
        String();
    };
    
    String::String()
    {
        assert(d_array_p = new char[d_size]); // 错误： assert 中应该只判断
                                              // 不要操作对象，否则会使 prod
                                              // 的版本和 debug 的版本行为不
                                              // 同。因为 prod 版 用 NDEBUG
                                              // 去掉了所有 assert.
    

        // 可以这样写
        d_array_p = new char[d_size];
        assert(d_array_p);
     }
